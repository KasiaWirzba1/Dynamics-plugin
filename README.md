# Dynamics with Gromacs — MoleQueue Integration Fork

This is a fork of the [Dynamics PyMOL plugin](https://github.com/makson96/Dynamics) developed by the Laboratory of Biomolecular Systems Simulation at the University of Gdansk (authors: Tomasz Makarewicz, Ajit B. Datta, Sara Boch Kminikowska, Manish Sud).

Original repository: https://github.com/makson96/Dynamics

The original plugin has been extended with remote job submission via MoleQueue as part of a Master's thesis at the Jagiellonian University.
